## Overview

CREAO is an AI agent platform company with 25 employees and 10 engineers. It builds tools that let users create autonomous AI agents through chat.

## Key Ideas

- Went fully AI-first: over 99% of production code is written by AI, with daily release cadence (previously every 4–6 weeks).
- Rebuilt its own platform using its own agents — "we built an agent platform, then the agents rebuilt it."
- Unified a fragmented multi-repo architecture into a single monorepo specifically for AI legibility.
- Runs a self-healing feedback loop: automated error detection → Claude-powered triage → auto-generated Linear tickets → fix verification → auto-close.
- Engineering org restructured into two roles: Architects (1–2 people designing systems and SOPs) and Operators (everyone else, handling investigation, review, and verification).
- 5 core engineers produce equivalent output to 100+ under the old model.
