## Overview

OpenAI is an AI research company and developer of Codex, a cloud-based coding agent used across the software development lifecycle. They are both a builder and practitioner of AI-native engineering.

## Key Ideas

- Developed the "Delegate / Review / Own" framework for structuring human-agent collaboration across each SDLC phase
- Internally report that development cycles that once took weeks now take days with Codex
- Routine tasks (dependency maintenance, feature flag cleanup, documentation) are fully delegated to Codex
- Advocate for MCP servers as the integration layer connecting agents to external tools (issue trackers, logging, deployment)
- Recommend AGENTS.md files as persistent agent configuration for behavior, guardrails, and workflow instructions
- Coined the term "harness engineering" (February 2026) for the practice of enabling agents to do useful work
